Status: SPEC_ONLY
Purpose: convert the review feedback for Gate P3 and Gate P6 into objective checks.

This file does not implement UI. It tells the integration owner which pitch actions can be marked LIVE, which must be RECORDED REAL RUN, and which remain ROADMAP.

## Endpoint Evidence Notes

Read-only inspection found these existing integration points:

- `GET /api/state` returns project_id, run_id, risk_map, runs_before, runs_after, verify, and live_scan.
- `POST /api/live-scan` writes live scan scope and runs the before path.
- `POST /api/install-guard` runs the after path, report, experiment, and verify.
- `POST /api/run` runs the full loop and verify.
- `POST /api/agent` runs the typed-ticket harness with an optional guard flag.

This automation must not call state-changing endpoints. It may only document how the integration owner should bind them.

## Truth Badge Gate

| Badge | Minimum evidence |
| --- | --- |
| LIVE | Current endpoint response or current file artifact opened during the demo |
| RECORDED REAL RUN | Saved artifact path and run_id visible to the judge |
| LOCAL FALLBACK | Local page/file used without claiming current live state |
| NOT CONFIGURED | Endpoint, artifact, or integration absent |
| PLACEHOLDER / ROADMAP | Planned capability without implementation evidence |

Fail-safe default: NOT CONFIGURED or LOCAL FALLBACK. Upgrade to LIVE only with evidence.

## Timing Gate

The three-minute timing gate is false until three measured rehearsals exist. A planned script, even a strong one, cannot set:

- `three_minute_timing_pass=true`
- `overall_pass=true`

Required measured rehearsal fields:

- date/time
- source label: LIVE, RECORDED REAL RUN, or LOCAL FALLBACK
- start and stop time
- duration
- artifact path or endpoint source
- visible failure moment timestamp
- fix/protection start timestamp
- after-proof completion timestamp

## Non-Duplication Gate

- ui-next integration

If a future reviewer asks for implementation, this branch should file or update an integration request rather than write UI files.
