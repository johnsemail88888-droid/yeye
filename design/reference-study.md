# Reference Study - Prototype Foundation

## Sources studied
- Linear: dense issue/editor surfaces with dimmer chrome, flat rows, hairline separators, and selected rows marked by a quiet accent.
- Raycast: command-first interaction model with a centered command surface and right-aligned shortcut hints.
- Vercel/Geist: restrained spacing, clipped surfaces, low-noise status labels, and sparse use of the primary accent.

## Moves copied into this increment
- Receded chrome: the rail is darker and quieter than the work area.
- Flat work rows: scan choices, findings, steps, and suggestions use compact full-width rows instead of decorative cards.
- Command spine: a first Cmd/Ctrl+K menu reaches mode switching and runtime scan focus.
- Honest labels: every sample finding/confidence label is visibly marked `MOCK`.
- Zoom safety: type and spacing use rem/clamp tokens, panel reflow uses container queries, and long values use `overflow-wrap:anywhere`.

## Not copied yet
- Mode 1 grade ring and collapsed risk accordions.
- Mode 2 squiggle + gutter dot + lightbulb triad.
- Resizable/collapsible suggestions panel and dockable preview.
